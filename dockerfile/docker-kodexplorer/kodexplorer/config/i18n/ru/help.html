Управление файлами

Выбор файла: радио, мышь бегущей строки, переложить эти выборы, Ctrl, выбранный случайным образом, клавиатуру вверх и вниз, домой, выбор конца.

Работа с файлами: После выбора файла, вы можете копировать, вырезать, удалять, просматривать свойства сжатия, переименовывать, открытый просмотр и другие операции ......

Загрузка файлов: Загрузить несколько файлов пакетной; html5 загрузки перетаскивание (плавно загружать окна, поддержку перетаскивания для папок)

Правая функция правой кнопкой файл, папку, справа, более сразу после операции выборов, рабочий стол, правый, правый дерево каталогов, правая граница ярлыки меню
(Select All - Копировать - Вырезать - Вставить - Удалить - переименование, комплект ......)

Браузер файлов: режим списка, режим значок, дважды щелкните на вложенные папки, операции в адресной строке, открыть папку записи противоположный рекорд (вперед и назад)

Поддержка перетаскивания: Проверьте сопротивление, сократить для достижения указанной функции папки

Клавиши быстрого доступа: удалить удалить, Ctrl + A Выделить все, Ctrl + C, чтобы скопировать, + X Вырезать, поиск файлов (поиск по содержимому) Упр

Предварительный просмотр файла

Предварительный просмотр файла: просматривать содержимое редактирования текстового файла и сохранение; HTML, SWF-файл предварительного просмотра,

Предварительный просмотр: автоматическая генерация миниатюр, слайд-шоу изображений;

Воспроизведение аудио: играть музыку онлайн и видео файлов, поддержка MP3, WMA, MID, AAC, WAV; MP4,

Видео: онлайн воспроизведение видео, поддерживаемые форматы: FLV, F4V, 3gp

офис: Интернет Preview, Поддерживаемые форматы: DOC, DOCX, PPT, PPTX, XLS, XLSX
